## Requirements
```
- MySql
- Go
```

## Install

```
go get github.com/eltonsouza/rv-test
```

## Configure database file
```json
{
    "database": {
        "host": "host",
        "user": "userbd",
        "password": "password",
        "port": "3306"
    }
}
```


## Running the project
```
go get ./...
go run main.go
```
